# first-pr-practice

Gyakorló repó az első GitHub pull request kipróbálásához.

## Miért jött létre?

Ez a repó azért készült, hogy egy egyszerű, biztonságos módon ki lehessen próbálni a GitHub-os
munkafolyamatot: branch létrehozása, módosítás, majd pull request nyitása.

## Használat

1. Hozz létre egy új branch-et
2. Végezz egy kis módosítást
3. Nyiss belőle egy pull requestet
